# 【完全AI生成】我的第一个使用CC做的项目——萌娘番茄钟

基于 **Electron** 的 Windows 桌面番茄钟应用，二次元风格界面，用于专注工作与休息节奏管理。

## 软件实现了什么

### 核心计时

- **番茄工作法流程**：专注 → 短休息 → … → 长休息，按你设定的「几个专注后进长休」自动切换阶段。
- **计时控制**：开始、暂停、重置、跳过；支持为当前专注填写**任务名称**（会写入记录）。
- **界面反馈**：大号倒计时、圆形进度环、当前阶段标签（专注 / 短休 / 长休）；显示今日已完成番茄数与当前周期进度。

### 设置与体验

- **时长可调**：专注、短休、长休时长（分钟），以及触发长休前的专注个数。
- **提示音**：结束提醒开关与音量；阶段结束时可选**系统自动通知**（主进程通知）。
- **窗口**：可选**置顶**；关闭窗口时行为与托盘联动（托盘图标可显示/隐藏窗口、退出）。
- **自动流转**：可选专注结束后**自动进入休息**阶段。

### 个性化

- **看板娘 / 吉祥物**：支持从本地导入图片作为界面角色；也可恢复默认形象。

### 记录与统计

- **会话历史**：每次专注/休息可记录完成情况（完成或中途跳过会区分）。
- **统计视图**：按**年**浏览日历热力图，点击某天查看当日各时段记录、任务名与专注总时长。

### 其他

- **单实例**：重复启动时会聚焦已有窗口，避免多开。
- **本地存储**：设置、吉祥物与历史记录保存在本机（`electron-store`），无需账号。

## 技术栈

- [Electron](https://www.electronjs.org/)（主进程 + 渲染进程，预加载脚本隔离 `nodeIntegration`）
- [electron-store](https://github.com/sindresorhus/electron-store) 持久化
- [electron-builder](https://www.electron.build/) 打包 Windows 便携版

## 开发与打包

```bash
npm install
npm start          # 开发运行
npm run build      # Windows 构建（输出见 dist）
npm run build:portable   # 便携版
```
